# Test plan

- **Caso feliz**:
  - Dado un traceback y pasos de repro, producir un reporte estructurado con Environment/Observed/Expected.

- **Errores**:
  - Si faltan pasos de repro: pedirlos sin inventar.
  - Si el traceback está incompleto: pedir el log completo.

- **Regresión**:
  - Aplicar el formato a 2 fallos distintos y confirmar consistencia del reporte.
