# Build y release (manual)

- Para recompilar el ejecutable:
  - Preferir el workflow `/build-installer-win` (si aplica) o el flujo definido por el proyecto.
  - Evitar comandos ad-hoc heredados de otros proyectos.

- Para preparar un paquete para compartir:
  - Preferir el workflow `/release-checklist`.
